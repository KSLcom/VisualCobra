Notes for developers wishing to build or modify Visual Cobra
------------------------------------------------------------

Prerequisites for building the solution:

* Visual Studio 2010 SP 1

You need Service Pack 1. Otherwise, you will get "project type (.csproj) is not
supported by this version of the application" errors.

* Visual Studio 2010 SP1 SDK (https://www.microsoft.com/en-us/download/confirmation.aspx?id=21835)
